AUTO PAN

True stereo auto pan with an option to average the stereo channels before panning.

Combo Box - Select the LFO waveform type
RATE - Set the LFO Rate
WET - Set wet/dry ratio of the effect
ACTIVE/BYPASS - Turn the effect on and off
TRU/AVG - TRU gives you true stereo panning and AVG averages the stereo signal to mono before panning.

Notes:
- You might experience audio "clicking" issues when using LFO waveforms aside from sine
